# Prisma PostgreSQL Project

This project is a backend application utilizing **Prisma** as an ORM and **PostgreSQL** as the database. The app provides user authentication, role-based access control, and other core functionalities for managing users and their roles.

## Features

- User Registration and Login
- Password Reset (Forgot & Reset)
- Role Management (Assign/Remove roles to/from users)
- Access Control based on Roles and Permissions

## Tech Stack

- **Prisma**: ORM tool for database
- **PostgreSQL**: Relational Database
- **Node.js**: Backend runtime
- **Express**: Web framework
- **JWT**: Authentication
- **Bcrypt**: Hashing passwords
- **Swagger**: API documentation

## Prerequisites

Before you start, ensure you have the following installed:

- **Node.js** (>=14.x)
- **PostgreSQL** database instance running locally or remotely

## Installation

- Install the dependecies

```bash
npm install
```

- Run the project

```bash
npm run dev
```
